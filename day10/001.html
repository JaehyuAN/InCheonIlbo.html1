html을 실행하는 방법 1. 로컬에 저장된 파일을 브라우저로 OPEN 2. 웹 서버에 html을 올리고 브라우저로 실행 live server는 개발용 장난감 서버 많이 사용되는 웹서버 : apache, nginx

1. 서버 : 사용자가 요청하면 서비스를 제공하는 프로그램 2. 웹서버(http 서버) 사용자가 요청하면 html을 보내주는 서버 프로그램 3. http는 웹에서 사용하는 통신 표준 4. http요청을 보내려면 서버의 IP주소와 포트를 알아야한다. 5. 사용자는 203.253.146.184 같은 IP를 기억하기 어렵다. 그래서 도메인주소를 사용 6. DNS: 도메인 이름을 IP주소로 바꿔주는 서비스 사용자가 도메인 이름을 입력하면 DNS가 IP로 변환한다.
